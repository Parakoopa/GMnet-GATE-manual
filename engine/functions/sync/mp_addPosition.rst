``mp_addPosition(groupname,interval)``
--------------------------------------

Description
~~~~~~~~~~~

Adds a new group of variables to be synced to this instance.

The variables are: \* x,y

Example
~~~~~~~

.. code-block:: gml

    ///Create Event
    mp_sync();
    mp_addPosition("pos",10*room_speed);;
